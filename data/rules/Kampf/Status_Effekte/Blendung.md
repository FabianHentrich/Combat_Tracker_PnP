Gegner wird geblendet. Effekte richten sich nach [[RANG]] der Blendung

| **Rang** | **Effekt**                                                |
| -------- | --------------------------------------------------------- |
| 1        | 1 Runde geblendet: [[Malus]] von -1 auf [[Angriffsprobe]] |
| 2        | 1 Runde geblendet: [[Malus]] von -2 auf [[Angriffsprobe]] |
| 3        | 1 Runde geblendet: [[Malus]] von -2 auf alle Aktionen     |
| 4        | 2 Runden geblendet: [[Malus]] von -2 auf alle Aktionen    |
| 5        | 2 Runden geblendet: [[Malus]] von -3 auf alle Aktionen    |
