# Eigenheiten

Eigenheiten sind spezielle Trade-Offs, welche die Charaktere erhalten können. Sie beinhalten immer einen Vorteil und einen Nachteil. Charaktere können eine Eigenheit von Beginn an haben oder sie während der Geschichte erhalten.
